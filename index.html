About Me

Hello! I'm Zeus Borrego, a junior software engineer with a passion for network programming, Magic: The Gathering, and solving problems. I love tackling challenging problems and am constantly seeking opportunities to learn and grow in the field of software development.

Projects

Mana Tomb

A Commander-focused Magic: The Gathering deck builder and card search tool. Built with Go and PostgreSQL, Mana Tomb emphasizes clean data pipelines, scalable architecture, and future expansion into deck analysis and AI-assisted feedback.

SwaTutor

Collaborated with a team of four to develop a web app that enables professors to assign tutors and students to request tutoring for courses. The backend was built using Python with Flask, while the frontend was developed with HTML, CSS, and JavaScript.

Ray-Marching Infinite Terrain Generator

Collaborated with a partner to build an infinite terrain generator using WebGL2 and JavaScript, running directly in the browser. Implemented noise functions, the Phong Reflection Model, and Ray Marching to generate a 3D world of islands.
